Visit the live site at: https://jackzzzzzzzzzzzz.github.io/Jack-Ruijie-Zhang/

## ✨ Features

- **Professional Design**: LinkedIn-inspired clean and modern interface
- **Responsive Layout**: Optimized for desktop, tablet, and mobile devices
- **Sections Include**:
  - About Me
  - Professional Experience
  - Technical Skills
  - Contact Information
- **LinkedIn Integration**: Direct links to LinkedIn profile
- **GitHub Pages Ready**: Automatically deployable via GitHub Pages

## 🛠️ Technologies Used

- HTML5
- CSS3 (with modern flexbox and grid layouts)
- Font Awesome icons
- Responsive design principles
- LinkedIn-inspired color scheme

## 🚀 Setup

To run locally:

1. Clone the repository
2. Open `index.html` in your browser, or
3. Run a local server: `python3 -m http.server 8000`
4. Visit `http://localhost:8000`
